# machado-advocacia-marketplace

Marketplace dos plugins jurídicos do escritório **Machado Advocacia**, no padrão **Adv-OS** (sistemas operacionais do advogado por área do direito).

## Plugins disponíveis

| Plugin | Área |
|--------|------|
| `consumidor-adv-os` | Direito do Consumidor e Bancário (35 skills, 14 commands) |

> Em breve, conforme migração/desenvolvimento: `previdenciario-adv-os`, `familia-sucessoes-adv-os`, `recuperacao-judicial-adv-os`, `usucapiao-adv-os`, `execucao-fiscal-adv-os`.

## Instalação (Claude Code / Cowork / Chat)

```
/plugin marketplace add gmachadogomesadvocacia-glitch/machado-advocacia-marketplace
/plugin install consumidor-adv-os@machado-advocacia-marketplace
```

Depois, configure o escritório com `/start-consumidor`.

## Atualização

```
/plugin marketplace update machado-advocacia-marketplace
```
