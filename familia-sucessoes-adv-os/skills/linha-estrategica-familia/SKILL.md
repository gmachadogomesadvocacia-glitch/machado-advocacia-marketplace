## 0. ESCOPO

Esta skill define o COMO do caso — a inteligência estratégica por trás das peças.
Não redige peças. Produz: análise de viabilidade + linha estratégica + plano de ação.

## 3. ACORDO vs. CONTENCIOSO

**Análise custo-benefício**:

| Fator | Acordo | Contencioso |
|-------|--------|-------------|
| Tempo | Semanas/meses | Meses/anos |
| Custo | Menor (custas + honorários) | Maior (todo o processo) |
| Controle do resultado | Alto | Baixo (depende do juiz) |
| Desgaste emocional | Menor | Maior |
| Privacidade | Maior | Menor (processo público) |
| Execução | Mais simples | Pode precisar de execução forçada |

**Quando recomendar acordo**:
- Partes têm disposição de negociar
- O resultado provável do contencioso é semelhante ao que se consegue no acordo
- Há filhos menores (prioridade: reduzir conflito)
- O cliente precisa de solução rápida

**Quando recomendar contencioso**:
- A outra parte age de má-fé ou está dilapidando patrimônio
- O acordo proposto é claramente desvantajoso para o cliente
- Há prova forte e jurisprudência favorável
- O cliente tem condições de suportar o litígio

## 4. SEQUÊNCIA ESTRATÉGICA DE ATOS

### 4.1 Para Requerente

```
FASE PRÉ-PROCESSUAL (antes de ajuizar):
1. Documentar o patrimônio (IR, matrículas, CRLVs) — antes que a outra parte venda
2. Fotografar bens comuns se possível
3. Guardar evidências digitais (conversas, e-mails) com data
4. Abrir conta bancária individual se usavam conta conjunta
5. Verificar se é caso de urgência → tutela antes de citar

FASE INICIAL:
6. Peticionar com tutela de urgência (se necessário)
7. Indicar todos os pedidos de uma vez (evitar inovação na fase recursal)
8. Requerer produção antecipada de prova contábil (se patrimônio suspeito)

FASE DE INSTRUÇÃO:
9. Rol de testemunhas estratégico (3-5 pessoas chave, não mais)
10. Requerer perícia (contábil, psicológica) com antecedência
11. Requerer expedição de ofícios (Receita Federal, DETRAN, B3, bancos)
```

### 4.2 Para Requerido

```
FASE INICIAL (prazo de 30 dias):
1. Ler toda a petição inicial e LISTAR cada pedido
2. Para cada pedido: tem base? Há prova? Qual a nossa versão?
3. Verificar reconvenção (vale a pena contra-atacar ou só defender?)
4. Verificar preliminares (competência, legitimidade, inépcia)
5. Documentar a versão do requerido com provas próprias

FASE DE INSTRUÇÃO:
6. Rol de testemunhas próprias
7. Contraditar provas documentais do requerente (questionar autenticidade ou irrelevância)
8. Requerer perícias favoráveis
```

## 6. CRONOGRAMA ESTIMADO

```
Ação de família padrão (TJ{{UF}} — {{CIDADE}}):
  Distribuição → Despacho inicial: 2-4 semanas
  Citação: 1-3 meses (depende de localização do réu)
  Contestação: 30 dias da citação
  Audiência de conciliação (art. 334 CPC): 30-60 dias após contestação
  Instrução: 3-9 meses
  Sentença: variável — 6-18 meses total
  Recurso (apelação): + 12-24 meses

Inventário extrajudicial: 2-6 semanas (se documentação completa)
Divórcio extrajudicial: 1-3 semanas
```
