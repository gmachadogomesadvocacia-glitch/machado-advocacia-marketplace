# LINHA ESTRATEGICA IMOBILIARIA

> Tier 1. Consolida trilateral + jurisprudencia e fixa **a via e a sequencia**. Decisao sempre documentada no CASO.md. Coerente com o polo (PA-10).

---

## 1. CRITERIOS DE DECISAO

| Criterio | Pergunta |
|----------|----------|
| **Notificacao previa** | Precisa constituir em mora / denunciar antes de ajuizar? |
| **Liminar/tutela** | Cabe? (despejo so nas hipoteses do art. 59 §1º + caucao — PA-06; possessoria de forca nova) |
| **Garantia/caucao** | Ha caucao a prestar (despejo liminar = 3 alugueis)? |
| **Custo** | Valor da causa, sucumbencia, honorarios |
| **Tempo** | Rito (sumario/comum), urgencia, prazo decadencial |
| **Foro** | Acao real = situacao do imovel (art. 47 CPC); locacao = eleicao/contrato |
| **Risco** | Prova suficiente? tese adversa forte? |

## 2. BIFURCACOES TIPICAS

**Notificar antes ou ajuizar ja?**
- Mora ja constituida (vencimento certo) → pode ajuizar. Denuncia vazia/cheia exige **notificacao previa** (prazo de desocupacao). Documentar a escolha.

**Possessoria x petitoria** (P4 / PA-05)
- Discussao de **posse** (esbulho, turbacao, data) → possessoria (reintegracao/manutencao/interdito). Discussao de **titulo/dominio** → reivindicatoria. **Nao misturar** — vedada *exceptio proprietatis* (art. 557 CPC).

**Despejo x cobranca**
- Quer o imovel de volta → despejo (pode cumular cobranca). Quer so o credito (locatario ja saiu) → acao de cobranca/execucao.

**Defesa na fiduciaria x anulatoria de leilao**
- Antes da consolidacao → purgar a mora (art. 26 §1º Lei 9.514). Apos leilao com vicio (intimacao, preco vil, excedente nao devolvido — art. 27 §4º) → anulatoria/consignatoria. Verificar a fase (PA-09).

**Renovatoria** — confirmar o **prazo decadencial** (1 ano a 6 meses antes do fim — art. 51 §5º, PA-07) ANTES de qualquer outra decisao; se perdido, redirecionar a estrategia.

## 3. SEQUENCIA RECOMENDADA

```
PASSO 0: notificacao extrajudicial? (S/N, prazo)
PASSO 1: peca principal (via escolhida) — foro: ...
PASSO 2: tutela/liminar? (cabimento + caucao)
PASSO 3: pedidos cumulados / valor da causa
PASSO 4: prova a produzir
```

## 4. SAIDA

```
TESE CENTRAL: ...
TESES SUBSIDIARIAS: ...
VIA / RITO: ...
FORO: ... (art. 47 se real)
NOTIFICACAO PREVIA: S/N
LIMINAR: cabivel? caucao?
RISCOS: ...
DECISAO DOCUMENTADA EM: CASO.md
```
Depois: producao da peca (porta da frente) + `estilo-juridico-imobiliario`, encerrando em `revisao-final-imobiliaria`.
