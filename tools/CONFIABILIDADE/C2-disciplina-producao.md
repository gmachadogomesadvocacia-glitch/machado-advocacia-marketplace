  **ficha de conferência** exige `cada R$ → fonte (calculos-*, data)`. (Regressão: regra

Antes de redigir, validar a base legal vigente NO FATO + as citações (cruzando com o
livro-razão — Camada 1). O Selo é o R1 da revisão e o protocolo P1 dos masters.
- ENFORÇADO POR: **C1** (livro-razão: citação fora dele = falha) + **C3** (R1/ficha registram o
  status das citações). Nos masters novos é o protocolo P1 explícito; nos 3 legados (família,

triagem → Selo P1 → análise documental → **cálculo** → produção → **revisão R1-R5** → entrega.
