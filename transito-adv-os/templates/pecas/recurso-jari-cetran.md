# CHASSI — RECURSO ADMINISTRATIVO (JARI 1ª inst. / CETRAN 2ª inst.)

NOTA DE USO
- Momento: APOS a Notificacao de Penalidade (NP) — a multa ja foi aplicada (nao confundir com defesa previa, PA-08).
- Um documento, DUAS variacoes: VARIACAO A = recurso a JARI (CTB 285); VARIACAO B = recurso ao CETRAN (CTB 288-290).
- Prazo em regra 30 dias (da NP para a JARI; da decisao da JARI para o CETRAN) — PA-05 preclusivo.
- Deposito previo VEDADO como condicao de admissibilidade (validar). Efeito suspensivo: so afirmar se validado (CTB 288 §2º).
- Coerencia com a defesa anterior (PA-10). Norma vigente na infracao (PA-04). Sumula consagrada + "(validar)" (PA-01).
- Texto corrido; CAIXA ALTA so destaque; docs "Doc. nº N" antes do protocolo; procuracao por ultimo.

Das nulidades (reiteracao e ampliacao).
